# 视觉小说翻译器 - 文本翻译与结果展示功能需求文档

## 1. 概述
本文档描述视觉小说翻译器的文本翻译及结果展示功能需求。该功能作为后台OCR模块的下游，接收OCR识别出的文本，进行翻译处理，并通过一个独立的悬浮窗口将翻译结果实时展示给用户。

## 2. 功能模块与流程

### 2.1 文本接收与去重模块
- **数据来源**：接收来自OCR模块输出的文本数据
- **去重判断**：
  - 将当前OCR识别出的文本与上一次成功翻译的文本进行比较
  - 比较逻辑：计算文本相似度（如Levenshtein距离或简单字符串对比）
  - 设定一个相似度阈值（例如差异超过20%视为新文本），超过阈值才触发翻译
  - 目的：避免同一句台词被反复翻译，减少API调用和界面闪烁
- **特殊情况**：
  - 如果OCR返回空文本，直接丢弃，不进入翻译流程
  - 如果文本仅包含标点符号或特殊字符，视为无效内容，不翻译

### 2.2 翻译引擎调用模块
- **触发条件**：文本去重模块判定为需要翻译的新文本
- **翻译参数**：
  - 源语言：继承自翻译设置界面的选择（日语/英语）
  - 目标语言：简体中文（固定）
  - 待翻译文本：OCR识别出的原文
- **引擎选择**：根据用户在翻译设置界面配置的翻译引擎（阿里云/Google翻译）和对应API密钥进行调用
- **调用方式**：异步调用，避免阻塞后续OCR处理流程
- **超时处理**：设置合理的超时时间（例如3-5秒），超时则本次翻译放弃，等待下一轮

### 2.3 翻译结果处理模块
- **结果接收**：接收翻译引擎返回的翻译结果
- **数据整合**：将原文与译文进行关联，形成完整的展示数据
- **异常处理**：
  - 翻译失败（API错误、网络问题）时，可显示错误提示（如“翻译失败”）
  - 连续失败记录日志，但不影响OCR模块继续运行

## 3. 翻译结果展示窗口

### 3.1 窗口基本属性
- **窗口类型**：独立悬浮窗口，始终置顶显示
- **背景样式**：半透明背景（例如透明度30%-50%），减少对游戏画面的遮挡
- **边框样式**：无边框或极细边框，仅在鼠标悬停时显示操作控件
- **初始位置**：默认显示在屏幕右下角或上次关闭时的位置

### 3.2 内容显示区域
- **原文显示**：
  - 以较小字体或较淡颜色显示（例如灰色）
  - 位置：译文上方或左侧，用户可辨识但不过分突出
- **译文显示**：
  - 以较大字体、醒目颜色显示（例如白色或亮黄色）
  - 支持多行文本自动换行
- **样式配置**（可后续扩展）：
  - 字体大小可调节
  - 字体颜色可配置
  - 背景透明度可调节

### 3.3 窗口交互操作
- **拖动移动**：
  - 鼠标在窗口任意位置按下左键并拖拽，可移动整个窗口
  - 拖动过程中窗口保持半透明，松开后固定在新位置
- **调整大小**：
  - 鼠标移动到窗口边缘时，显示大小调整光标
  - 拖拽边缘可拉长或拉宽窗口
  - 窗口内容根据新尺寸自动重排
- **右键菜单**（可选）：
  - 提供“复制译文”、“复制原文”、“隐藏窗口”、“退出程序”等选项

### 3.4 窗口状态管理
- **显示/隐藏**：支持快捷键或系统托盘图标控制窗口的显示与隐藏
- **自动隐藏**：可配置鼠标离开窗口后自动半透明或缩小为标题栏
- **多显示器支持**：窗口可以在任意显示器上移动和显示

## 4. 整体数据流程

---------------------------------------------
| 文本翻译与展示逻辑流程                      |
---------------------------------------------
| 从OCR模块接收文本数据 → 当前文本             |
|      ↓                                       |
| 与上一次翻译的文本进行相似度对比               |
|      ↓                                       |
| 判断：差异是否超过阈值？                       |
|      ├─ 否 → 丢弃，等待下一条OCR结果          |
|      └─ 是 → 进入翻译流程                      |
|           ↓                                   |
|          根据当前选中的翻译引擎和API密钥        |
|          调用对应的翻译API (阿里云/Google)      |
|           ↓                                   |
|          等待翻译结果返回                       |
|           ↓                                   |
|          判断：翻译是否成功？                    |
|           ├─ 否 → 在悬浮窗显示错误提示          |
|           └─ 是 → 整合原文与译文数据            |
|                ↓                               |
|               更新悬浮窗显示内容                 |
|                ↓                               |
|               记录本次翻译的文本作为下一次去重依据 |
|                ↓                               |
|        等待下一条OCR结果                        |
---------------------------------------------

## 5. 配置参数（可后续在设置界面提供）

- **文本去重阈值**：默认值建议20%，支持调节
- **翻译超时时间**：默认3秒，支持调节
- **悬浮窗默认透明度**：默认40%，支持调节
- **字体大小**：原文默认12px，译文默认16px，支持调节
- **显示原文开关**：可选择是否在悬浮窗中显示原文
