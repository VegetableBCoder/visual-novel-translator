# 视觉小说翻译器 - 运行参数配置界面需求文档

## 1. 概述
本文档描述视觉小说翻译器的第四个界面——运行参数配置界面的功能与交互需求。该界面在用户完成截图区域选择后显示，用于配置OCR识别、文本去重及翻译结果展示的具体参数。完成配置后，用户可点击“开始翻译”启动自动翻译流程。

## 2. 界面布局与功能

### 2.1 窗口标题
- 标题文字：**运行参数配置**

### 2.2 引导文案
- 文案内容：*“请配置OCR识别和翻译结果的显示参数”*
- 位置：界面顶部，配置项上方

### 2.3 OCR相关参数配置区域

- **OCR截图间隔**
  - 标签文字：OCR截图间隔（毫秒）
  - 控件类型：数字输入框 + 滑块（可选）
  - 取值范围：100 - 10000 毫秒
  - 默认值：1000
  - 步进值：100
  - 说明文字：*“数值越小识别越频繁，性能占用越高”*

- **OCR图片相似度判断阈值**
  - 标签文字：图片相似度阈值（0-100）
  - 控件类型：数字输入框 + 滑块
  - 取值范围：0 - 100
  - 默认值：80（或根据pHash特性建议值）
  - 说明文字：*“数值越高，图片变化越小越容易触发OCR”*
  - 补充说明：0表示每次都识别，100表示只有完全不同的图片才识别

### 2.4 文本处理参数配置区域

- **文本去重相似度阈值**
  - 标签文字：文本去重阈值（0-100）
  - 控件类型：数字输入框 + 滑块
  - 取值范围：0 - 100
  - 默认值：80
  - 说明文字：*“数值越高，文本变化越小越容易触发翻译”*
  - 补充说明：0表示每次都翻译，100表示只有完全不同的文本才翻译

### 2.5 翻译结果展示配置区域

- **字体选择**
  - 标签文字：译文字体
  - 控件类型：下拉选择框
  - 选项：系统已安装的所有中文字体（通过系统API枚举）
  - 默认值：微软雅黑 或 宋体

- **字体大小**
  - 标签文字：字体大小
  - 控件类型：数字输入框 + 滑块
  - 取值范围：8 - 72
  - 默认值：16
  - 步进值：1

- **字体颜色**
  - 标签文字：字体颜色
  - 控件类型：颜色选择按钮
  - 交互：点击后弹出标准颜色选择对话框
  - 默认值：白色（#FFFFFF）

- **背景透明度**
  - 标签文字：窗口透明度（%）
  - 控件类型：数字输入框 + 滑块
  - 取值范围：0 - 100（0%完全透明，100%完全不透明）
  - 默认值：40
  - 说明文字：*“数值越高，悬浮窗背景越不透明”*

### 2.6 底部操作按钮

- **上一步**
  - 功能：返回截图区域选择界面，当前配置不保存
  - 初始状态：可点击

- **开始翻译**
  - 功能：保存所有配置，启动后台OCR和翻译流程，打开翻译结果悬浮窗
  - 样式：突出显示（如蓝色高亮）

### 2.7 运行中状态（点击开始翻译后）

- **界面变化**：
  - “开始翻译”按钮变为 **“暂停翻译”**
  - “上一步”按钮变为 **不可点击状态**（置灰）
  - 所有配置项变为 **只读/不可修改** 状态（置灰）

- **暂停翻译状态**：
  - 点击“暂停翻译”后，后台OCR和翻译流程暂停
  - 按钮文字变回 **“开始翻译”**
  - “上一步”按钮恢复为 **可点击状态**
  - 所有配置项恢复为 **可修改** 状态

## 3. 交互逻辑补充

### 3.1 配置验证
- 所有数值输入框应有边界检查，超出范围时自动修正为最近的有效值
- 字体选择下拉框应实时预览选中的字体效果（可选）

### 3.2 状态保持
- 关闭界面时（如点击右上角X），应视为取消操作，不保存配置

### 3.3 运行中状态提示
- 进入运行状态后，可在界面底部显示状态文字：*“翻译运行中，参数已锁定”*
- 暂停时显示：*“翻译已暂停”*
