# 视觉小说翻译器 - 翻译设置界面需求文档

## 1. 概述
本文档描述视觉小说翻译器的首个界面——翻译设置界面的功能与交互需求。该界面在软件启动时显示，用户完成基础配置后点击“下一步”进入主程序。

## 2. 界面布局与功能

### 2.1 窗口标题
- 标题文字：**翻译设置**

### 2.2 语言设置区域
- **源语言（待翻译语言）**
  - 控件类型：下拉选择框
  - 选项：日语、英语
  - 默认值：日语
  - 状态：所有选项固定，不可自定义输入

- **目标语言（翻译结果语言）**
  - 控件类型：下拉选择框
  - 选项：简体中文
  - 默认值：简体中文
  - 状态：选项固定，不可更改

### 2.3 翻译引擎选择区域
- **引擎选择**
  - 控件类型：单选框
  - 选项1：阿里云翻译
  - 选项2：Google翻译
  - 默认值：阿里云翻译

- **引擎配置输入**
  - 根据选中的引擎显示对应的配置项

  **阿里云翻译配置项：**
  - AccessKey ID（文本输入框）
  - AccessKey Secret（文本输入框，密码模式）
  - 验证按钮：点击后尝试用当前密钥进行测试翻译，反馈“验证成功”或“验证失败，请检查密钥”

  **Google翻译配置项：**
  - API Key（文本输入框，密码模式）
  - 验证按钮：点击后尝试用当前密钥进行测试翻译，反馈“验证成功”或“验证失败，请检查密钥”

- **帮助信息**
  - 设计：每个引擎配置区域右侧放置一个问号图标（ⓘ）
  - 交互：鼠标悬停或点击时显示提示文本
  - 阿里云提示文本：*“AccessKey可在阿里云控制台-访问管理页面获取”*
  - Google提示文本：*“API Key可在Google Cloud控制台-凭据页面获取；国内使用需自行解决网络问题”*

### 2.4 底部操作按钮
- **取消**
  - 功能：关闭窗口，不保存任何更改

- **下一步**
  - 功能：保存当前配置，关闭设置窗口，进入主程序界面
  - 样式：突出显示（如蓝色高亮）

## 3. 交互逻辑补充
- 所有配置项在关闭窗口前应持久化，下次打开软件时自动加载上次的配置
- 密钥输入框支持粘贴操作
- 验证按钮应显示加载状态，防止重复点击
