# 翻译设置界面 - 详细技术设计文档

## 1. 界面概述
翻译设置界面是软件启动后的第一个界面，用户在此完成最基本的翻译参数配置，包括：
- 选择源语言（日语/英语）
- 选择目标语言（简体中文，固定）
- 选择翻译引擎（阿里云/Google）
- 配置对应引擎的API密钥
- 点击"下一步"进入窗口选择界面

## 2. 技术实现要点

### 2.1 界面容器与布局
- 使用 `QWidget` 作为界面主容器，设置为窗口的中心部件。
- 采用 `QVBoxLayout` 垂直布局，内部嵌套多个 `QGroupBox` 进行区域划分：
  - 语言设置区域（QGroupBox）
  - 翻译引擎区域（QGroupBox）
- 底部按钮区域使用 `QHBoxLayout` 放置"取消"和"按钮"按钮。
- 所有控件采用固定宽度或大小策略，确保界面在不同分辨率下显示稳定。

### 2.2 语言选择控件
- 源语言：使用 `QComboBox` 下拉框，选项固定为"日语"和"英语"。默认选中"日语"。
- 目标语言：使用 `QLabel` 显示"简体中文"，或使用只读的 `QComboBox` 仅包含一个选项。直观上让用户知道不可修改。
- 语言区域无需互换按钮，简洁呈现。

### 2.3 翻译引擎选择与配置
- 引擎选择：使用 `QButtonGroup` 管理两个 `QRadioButton`，分别对应"阿里云翻译"和"Google翻译"。默认选中阿里云。
- 引擎配置区域：使用 `QStackedWidget` 实现根据选中的引擎动态切换不同的配置面板。
  - 阿里云面板：包含两个 `QLineEdit`，分别用于输入 AccessKey ID 和 AccessKey Secret（后者设置 `echoMode` 为 `QLineEdit::Password`）。
  - Google面板：包含一个 `QLineEdit` 用于输入 API Key（密码模式）。
- 每个面板内放置一个"验证"按钮 (`QPushButton`) 和一个帮助问号标签 (`QLabel` 显示为问号图标)。
- 帮助信息：点击问号图标弹出 `QToolTip` 或一个小型 `QMessageBox` 显示提示文本。

### 2.4 底部按钮
- "取消"：关闭窗口，不保存任何更改。
- "下一步"：保存当前配置并切换到窗口选择界面。初始状态始终可点击（因为默认值已存在），但若用户输入了不合法的密钥，可在验证失败时给出警告，但不应阻止下一步。

## 3. 配置管理架构（依赖倒置设计）

### 3.1 配置接口定义 (controller/config_interface.py)

定义抽象接口，使得界面层依赖于接口而非具体实现：

```python
from abc import ABC, abstractmethod
from typing import Any, Dict

class IConfigManager(ABC):
    """配置管理器抽象接口"""

    @abstractmethod
    def load_config(self) -> Dict[str, Any]:
        """加载完整配置，返回字典"""
        pass

    @abstractmethod
    def save_config(self, config: Dict[str, Any]) -> bool:
        """保存配置到持久化存储"""
        pass

    @abstractmethod
    def get(self, key: str, default: Any = None) -> Any:
        """获取配置项的值，支持点号分隔的路径如 'language.source'"""
        pass

    @abstractmethod
    def set(self, key: str, value: Any) -> None:
        """设置配置项的值，支持点号分隔的路径"""
        pass

    @abstractmethod
    def ensure_defaults(self) -> None:
        """确保默认配置存在"""
        pass
```

### 3.3 配置辅助类 (controller/config_helper.py)

临时实现，基于 JSON 文件的简单配置管理：

```python
import json
import os
from controller.config_interface import IConfigManager
from typing import Any, Dict

class ConfigHelper(IConfigManager):
    """配置管理器临时实现，基于 JSON 文件"""

    DEFAULT_CONFIG = {
        "version": "1.0",
        "language": {
            "source": "ja",
            "target": "zh"
        },
        "translation": {
            "engine": "alibaba",
            "alibaba": {
                "access_key_id": "",
                "access_key_secret": ""
            },
            "google": {
                "api_key": ""
            }
        }
    }

    def __init__(self, config_path: str = "data/config.json"):
        self.config_path = config_path
        self._config = {}
        self._ensure_config_dir()
        self.load_config()

    def _ensure_config_dir(self):
        """确保配置文件目录存在"""
        os.makedirs(os.path.dirname(self.config_path), exist_ok=True)

    def load_config(self) -> Dict[str, Any]:
        """从文件加载配置"""
        if not os.path.exists(self.config_path):
            self._config = self.DEFAULT_CONFIG.copy()
            self.save_config(self._config)
            return self._config

        try:
            with open(self.config_path, 'r', encoding='utf-8') as f:
                self._config = json.load(f)
            self.ensure_defaults()
            return self._config
        except (json.JSONDecodeError, IOError):
            self._config = self.DEFAULT_CONFIG.copy()
            return self._config

    def save_config(self, config: Dict[str, Any]) -> bool:
        """保存配置到文件"""
        try:
            with open(self.config_path, 'w', encoding='utf-8') as f:
                json.dump(config, f, indent=2, ensure_ascii=False)
            self._config = config.copy()
            return True
        except IOError:
            return False

    def get(self, key: str, default: Any = None) -> Any:
        """获取嵌套配置值"""
        keys = key.split('.')
        value = self._config
        for k in keys:
            if isinstance(value, dict) and k in value:
                value = value[k]
            else:
                return default
        return value

    def set(self, key: str, value: Any) -> None:
        """设置嵌套配置值"""
        keys = key.split('.')
        config = self._config
        for k in keys[:-1]:
            if k not in config:
                config[k] = {}
            config = config[k]
        config[keys[-1]] = value

    def ensure_defaults(self) -> None:
        """确保必要的配置项存在"""
        def deep_update(target: Dict, config: Dict):
            for k, v in config.items():
                if k not in target:
                    target[k] = v
                elif isinstance(v, dict):
                    deep_update(target[k], v)
        deep_update(self._config, self.DEFAULT_CONFIG)
```

## 4. 交互逻辑

### 4.1 引擎切换

当用户点击不同的引擎单选框时，QStackedWidget 切换到对应的配置面板。

切换后，之前面板输入的密钥不会被清除，以便用户再次切回时保留。

### 4.2 密钥验证（同步方式）

点击"验证"按钮时，执行同步验证流程：

1. 按钮文本变为"验证中..."并禁用，防止重复点击
2. 读取当前面板的密钥值
3. 调用 `_validate_key_sync()` 方法执行同步 API 调用
4. 验证完成后，恢复按钮状态
5. 通过 QMessageBox 提示验证结果（成功或失败）

同步验证实现：

### 4.3 下一步

收集当前所有控件的值：
- 源语言下拉框当前文本 → 映射为代码值（"ja"/"en"）。
- 引擎单选框选中状态 → 确定引擎类型。
- 根据引擎类型读取对应输入框的密钥。

将配置合并到配置对象中，调用配置管理器的 `save_config()` 持久化。

发射信号或调用主窗口的方法切换到窗口选择界面。

### 4.4 取消

直接关闭窗口，若用户尚未进入后续界面，则退出程序；若已进入后续界面再返回，则取消返回上一界面。本界面作为启动第一界面，取消通常意味着退出应用。

## 5. 界面类设计

### 5.1 TranslationSettingsWidget 类定义

```python
from PyQt5.QtWidgets import (QWidget, QVBoxLayout, QHBoxLayout, QGroupBox,
                           QLabel, QComboBox, QRadioButton,
                           QPushButton, QLineEdit, QStackedWidget,
                           QMessageBox, QFrame)
from PyQt5.QtCore import Qt, Signal
from controller.config_interface import IConfigManager

class TranslationSettingsWidget(QWidget):
    """翻译设置界面"""

    # 定义信号，用于与主窗口通信
    next_signal = Signal()        # 用户点击下一步
    cancel_signal = Signal()       # 用户点击取消

    def __init__(self, config_manager: IConfigManager, parent=None):
        super().__init__(parent)
        self.config_manager = config_manager

        self._init_ui()
        self._load_config()
        self._connect_signals()

    def _init_ui(self):
        """初始化界面"""
        self.setWindowTitle("翻译设置")
        layout = QVBoxLayout(self)

        # 添加各组件
        layout.addWidget(self._create_language_group())
        layout.addWidget(self._create_engine_group())

        # 底部按钮
        button_layout = QHBoxLayout()
        self.cancel_btn = QPushButton("取消")
        self.next_btn = QPushButton("下一步")
        button_layout.addWidget(self.cancel_btn)
        button_layout.addWidget(self.next_btn)
        button_layout.addStretch()
        layout.addLayout(button_layout)

    def _create_language_group(self) -> QGroupBox:
        """创建语言设置组"""
        group = QGroupBox("语言设置")
        layout = QVBoxLayout()

        # 源语言
        source_layout = QHBoxLayout()
        source_layout.addWidget(QLabel("源语言："))
        self.source_lang_combo = QComboBox()
        self.source_lang_combo.addItems(["日语", "英语"])
        source_layout.addWidget(self.source_lang_combo)
        source_layout.addStretch()
        layout.addLayout(source_layout)

        # 目标语言
        target_layout = QHBoxLayout()
        target_layout.addWidget(QLabel("目标语言："))
        target_label = QLabel("简体中文")
        target_label.setStyleSheet("color: gray;")
        target_layout.addWidget(target_label)
        target_layout.addStretch()
        layout.addLayout(target_layout)

        group.setLayout(layout)
        return group

    def _create_engine_group(self) -> QGroupBox:
        """创建翻译引擎配置组"""
        group = QGroupBox("翻译引擎")
        layout = QVBoxLayout()

        # 引擎选择
        self.engine_group = QButtonGroup(self)
        self.alibaba_radio = QRadioButton("阿里云翻译")
        self.google_radio = QRadioButton("Google翻译")
        self.engine_group.addButton(self.alibaba_radio, 0)
        self.engine_group.addButton(self.google_radio, 1)

        engine_layout = QHBoxLayout()
        engine_layout.addWidget(self.alibaba_radio)
        engine_layout.addWidget(self.google_radio)
        engine_layout.addStretch()
        layout.addLayout(engine_layout)

        # 配置面板堆叠
        self.stacked_widget = QStackedWidget()
        self.stacked_widget.addWidget(self._create_alibaba_panel())
        self.stacked_widget.addWidget(self._create_google_panel())
        layout.addWidget(self.stacked_widget)

        group.setLayout(layout)
        return group

    def _create_alibaba_panel(self) -> QWidget:
        """创建阿里云配置面板"""
        panel = QWidget()
        layout = QVBoxLayout()

        # AccessKey ID
        id_layout = QHBoxLayout()
        id_layout.addWidget(QLabel("AccessKey ID："))
        self.alibaba_id_edit = QLineEdit()
        id_layout.addWidget(self.alibaba_id_edit)
        layout.addLayout(id_layout)

        # AccessKey Secret
        secret_layout = QHBoxLayout()
        secret_layout.addWidget(QLabel("AccessKey Secret："))
        self.alibaba_secret_edit = QLineEdit()
        self.alibaba_secret_edit.setEchoMode(QLineEdit.Password)
        secret_layout.addWidget(self.alibaba_secret_edit)
        layout.addLayout(secret_layout)

        # 验证按钮和帮助
        btn_layout = QHBoxLayout()
        self.alibaba_verify_btn = QPushButton("验证")
        help_label = QLabel("ⓘ")
        help_label.setToolTip("AccessKey可在阿里云控制台-访问管理页面获取")
        help_label.setStyleSheet("color: blue; cursor: help;")
        btn_layout.addWidget.addWidget(self.alibaba_verify_btn)
        btn_layout.addWidget(help_label)
        btn_layout.addStretch()
        layout.addLayout(btn_layout)

        panel.setLayout(layout)
        return panel

    def _create_google_panel(self) -> QWidget:
        """创建 Google 配置面板"""
        panel = QWidget()
        layout = QVBoxLayout()

        # API Key
        key_layout = QHBoxLayout()
        key_layout.addWidget(QLabel("API Key："))
        self.google_key_edit = QLineEdit()
        self.google_key_edit.setEchoMode(QLineEdit.Password)
        key_layout.addWidget(self.google_key_edit)
        layout.addLayout(key_layout)

        # 验证按钮和帮助
        btn_layout = QHBoxLayout()
        self.google_verify_btn = QPushButton("验证")
        help_label = QLabel("ⓘ")
        help_label.setToolTip("API Key可在Google Cloud控制台-凭据页面获取")
        help_label.setStyleSheet("color: blue; cursor: help;")
        btn_layout.addWidget(self.google_verify_btn)
        btn_layout.addWidget(help_label)
        btn_layout.addStretch()
        layout.addLayout(btn_layout)

        panel.setLayout(layout)
        return panel

    def _connect_signals(self):
        """连接信号"""
        self.engine_group.buttonClicked.connect(self._on_engine_changed)
        self.alibaba_verify_btn.clicked.connect(self._on_verify_alibaba)
        self.google_verify_btn.clicked.connect(self._on_verify_google)
        self.cancel_btn.clicked.connect(self.cancel_signal.emit)
        self.next_btn.clicked.connect(self._on_next)

    def _load_config(self):
        """加载配置到界面"""
        config = self.config_manager.load_config()

        # 加载源语言
        source = config.get("language.source", "ja")
        self.source_lang_combo.setCurrentIndex(0 if source == "ja" else 1)

        # 加载引擎选择
        engine = config.get("translation.engine", "alibaba")
        if engine == "google":
            self.google_radio.setChecked(True)
        else:
            self.alibaba_radio.setChecked(True)

        # 加载密钥
        self.alibaba_id_edit.setText(config.get("translation.alibaba.access_key_id", ""))
        self.alibaba_secret_edit.setText(config.get("translation.alibaba.access_key_secret", ""))
        self.google_key_edit.setText(config.get("translation.google.api_key", ""))

    def _save_config(self):
        """保存界面配置"""
        # 保存源语言
        source_code = "ja" if self.source_lang_combo.currentIndex() == 0 else "en"
        self.config_manager.set("language.source", source_code)

        # 保存引擎选择
        engine = "google" if self.google_radio.isChecked() else "alibaba"
        self.config_manager.set("translation.engine", engine)

        # 保存密钥
        self.config_manager.set("translation.alibaba.access_key_id",
                            self.alibaba_id_edit.text())
        self.config_manager.set("translation.alibaba.access_key_secret",
                            self.alibaba_secret_edit.text())
        self.config_manager.set("translation.google.api_key",
                            self.google_key_edit.text())

        # 持久化
        self.config_manager.save_config(self.config_manager.load_config())

    def _on_engine_changed(self, button):
        """引擎切换处理"""
        if button == self.alibaba_radio:
            self.stacked_widget.setCurrentIndex(0)
        elif button == self.google_radio:
            self.stacked_widget.setCurrentIndex(1)

    def _on_verify_alibaba(self):
        """验证阿里云密钥"""
        self.alibaba_verify_btn.setEnabled(False)
        self.alibaba_verify_btn.setText("验证中...")

        if self._validate_alibaba_key_sync(
            self.alibaba_id_edit.text().strip(),
            self.alibaba_secret_edit.text().strip()
        ):
            QMessageBox.information(self, "验证成功", "阿里云密钥验证成功！")
        else:
            QMessageBox.warning(self, "验证失败", "验证失败，请检查密钥或网络连接。")

        self.alibaba_verify_btn.setEnabled(True)
        self.alibaba_verify_btn.setText("验证")

    def _on_verify_google(self):

        self.google_verify_btn.setText("验证中...")

            self.google_key_edit.text().strip()

        else:

        self.google_verify_btn.setEnabled(True)
        self.google_verify_btn.setText("验证")

    def _validate_alibaba_key_sync(self, key_id: str, key_secret: str) -> bool:
        """同步验证阿里云密钥"""

    def _validate_google_key_sync(self, api_key: str) -> bool:

    def _on_next(self):

        self._save_config()
```

## 6. 配置加载与保存

### 6.1 初始化加载

在界面显示前，通过 `ConfigManager.load_config()` 获取完整配置。

根据配置设置：
- 源语言：若存在且为 `"en"`，则下拉框选英语，否则默认日语。
- 引擎：根据 `translation.engine` 选中对应单选框，并切换到对应面板。
- 密钥：填充到对应输入框（若存在）。

### 6.2 保存时机

点击"下一步"时，将当前界面数据写入配置对象，并调用 `ConfigManager.save_config()` 持久化。为保证配置不丢失，建议立即保存。

## 7. 异常处理

- 配置加载失败（如 JSON 损坏）：使用默认配置填充界面。
- 验证按钮调用的 API 超时或网络错误：捕获异常，提示用户检查网络或密钥格式。
- 输入框内容包含特殊字符：API 密钥通常无限制，但可简单校验非空。

## 8. 后续扩展考虑

- 如需增加更多源语言，只需扩展下拉框选项和语言映射表。
- 如需增加新翻译引擎，在 `QStackedWidget` 中添加新面板，并在引擎枚举和配置结构中增加对应项。

## 9. 与其他模块的交互

- 依赖 `IConfigManager` 接口，通过 `ConfigHelper` 临时实现进行配置管理。
- 通过主窗口的界面切换控制器（如一个堆叠窗口控制器）触发界面跳转。
- 验证功能使用 `requests` 库直接调用 API 进行测试。
