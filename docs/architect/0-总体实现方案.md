# 视觉小说翻译器 - 总体技术架构设计文档

## 1. 文档说明
本文档基于前期完成的功能需求文档和技术选型讨论，进行技术选型总结和整体架构设计。后续将基于此架构，对每个模块进行详细的技术设计。

## 2. 技术选型总结

### 2.1 开发语言与核心框架
| 组件      | 选型                 | 说明                   |
|---------|--------------------|----------------------|
| 编程语言    | Python 3.14        | 开发效率高，AI辅助效果好        |
| GUI框架   | PyQt5              | 稳定成熟，社区资源丰富          |
| Win32操作 | pywin32            | 功能全面，直接调用Windows API |
| 图像处理    | Pillow + imagehash | 轻量够用，pHash直接支持       |
| 数值计算    | numpy              | 可选，用于图像数组操作          |

### 2.2 核心功能模块选型
| 模块    | 选型                  | 说明                            |
|-------|---------------------|-------------------------------|
| OCR引擎 | PaddleOCR           | 日语支持好，本地免费，支持竖排文字，无需预处理直接输入原图 |
| 翻译引擎  | 阿里云API / Google免费节点 | 两者都要支持，根据用户配置                 |
| 文本去重  | Levenshtein编辑距离     | 计算文本相似度，阈值70-85可调             |
| 配置存储  | JSON / QSettings    | 保存用户配置和API密钥                  |

```json
{
  "version": "1.0",
  "language": {
    "source": "ja",        // ja:日语, en:英语
    "target": "zh"         // 固定简体中文
  },
  "translation": {
    "engine": "alibaba",    // alibaba 或 google
    "alibaba": {
      "access_key_id": "",
      "access_key_secret": ""
    },
    "google": {
      "api_key": ""
    }
  },
  "window": {
    "hwnd": 123456,        // 窗口句柄
    "title": "",           // 窗口标题（备忘）
    "region": {            // 相对比例 (0-1)
      "left": 0.25,
      "top": 0.70,
      "right": 0.75,
      "bottom": 0.85
    }
  },
  "ocr": {
    "interval_ms": 1000,   // 截图间隔
    "image_threshold": 10,  // 图片相似度阈值 (pHash汉明距离)
    "text_threshold": 75    // 文本相似度阈值 (0-100)
  },
  "display": {
    "font_family": "微软雅黑",
    "font_size": 16,
    "font_color": "#FFFFFF",
    "bg_opacity": 40,       // 0-100
    "show_original": true   // 是否显示原文
  }
}
```

## 6. 线程模型

```commandline
| 线程模型设计 (三线程) |
| |
| 主线程 (UI线程) |
| ┌─────────────────────────────────────┐ |
| │ - PyQt5 事件循环 │ |
| │ - 用户界面响应 │ |
| │ - 悬浮窗更新 │ |
| │ - 接收工作线程信号更新UI │ |
| └─────────────────────────────────────┘ |
| ↑ |
| │ 信号/槽 (线程安全) |
| ↓ |
| 工作线程1 (截图线程) |
| ┌─────────────────────────────────────┐ |
| │ - 定时循环 (间隔可配置) │ |
| │ - 窗口截图 (pywin32) │ |
| │ - 区域裁剪 (Pillow) │ |
| │ - 计算pHash (imagehash) │ |
| │ - 图片级去重判断 │ |
| │ - (变化时) 发信号给OCR线程 │ |
| └─────────────────────────────────────┘ |
| ↓ |
| 工作线程2 (OCR/翻译线程) |
| ┌─────────────────────────────────────┐ |
| │ - 接收截图信号 │ |
| │ - OCR识别 (PaddleOCR, 直接原图) │ |
| │ - 文本级去重 (Levenshtein) │ |
| │ - (新文本) 调用翻译API │ |
| │ - 格式化结果 │ |
| │ - 发信号给主线程更新悬浮窗 │ |
| └─────────────────────────────────────┘ |
| |
| 说明: |
| - 截图线程独立循环，保证截图频率稳定 |
| - OCR/翻译线程处理耗时操作，不阻塞截图 |
| - 线程间通过Qt信号/槽传递数据 |
| - 使用QMutex保护共享数据（如配置） |
| |
```

## 7. 模块接口定义（概要）

### 7.1 配置管理器

```
class ConfigManager:
    def load_config() -> dict
    def save_config(config: dict)
    def get(key: str) -> any
    def set(key: str, value: any)
```

### 7.2 窗口管理器

```
class WindowManager:
    def enum_windows() -> list[dict]  # [{hwnd, title, class}]
    def get_window_rect(hwnd) -> tuple
    def is_window_valid(hwnd) -> bool
```

### 7.3 截图服务

```
class CaptureService:
    def capture_window(hwnd) -> PIL.Image
    def capture_region(hwnd, region_ratio) -> PIL.Image
```

### 7.4 图像处理器

```
class ImageProcessor:
    def crop(image, region_ratio) -> PIL.Image
    def calculate_hash(image) -> imagehash.ImageHash  # 内部自动转灰度
    def is_image_changed(hash1, hash2, threshold) -> bool
```

### 7.5 OCR服务

```
class OCRService:
    def __init__(model_dir=None)
    def recognize(image) -> dict  # 直接输入PIL原图，返回{"text": str, "confidence": float}
```

### 7.6 文本去重器

```
class TextDeduplicator:
    def __init__(threshold=75)
    def is_similar(text1, text2) -> bool  # 基于Levenshtein距离
    def should_translate(new_text) -> bool  # 维护上一次文本状态
    def reset()
```

### 7.7 翻译服务
```
class TranslationService:
    def __init__(engine, api_key, secret=None)
    def translate(text, source_lang, target_lang) -> str
```

### 7.8 任务调度器

```
class Scheduler:
    def start()
    def pause()
    def resume()
    def stop()
    def set_interval(ms)
```
