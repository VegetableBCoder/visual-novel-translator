# 运行参数配置界面 - 详细技术设计文档

## 1. 界面概述
运行参数配置界面是软件启动后的第四个界面，也是最后一个配置界面。用户在此完成OCR参数、文本去重参数以及翻译结果悬浮窗的显示样式配置。点击“开始翻译”后，所有配置被保存，后台线程启动，界面进入运行中状态（参数锁定）。点击“暂停翻译”可暂停后台任务并解锁界面。

## 2. 界面容器与布局

### 2.1 主容器
- 使用 `QWidget` 作为界面主容器，设置为 `QVBoxLayout` 垂直布局。
- 顶部放置引导文案 `QLabel`。
- 中间依次放置三个 `QGroupBox` 分组框，分别对应OCR参数、文本处理参数、翻译结果显示设置。
- 每个分组框内部使用 `QGridLayout` 或 `QFormLayout` 进行标签与控件的对齐。
- 底部放置一个水平布局，包含状态标签 `QLabel` 和两个按钮（上一步、开始/暂停）。

### 2.2 控件说明

#### 2.2.1 OCR参数区域
- **OCR截图间隔**：
  - 使用 `QSpinBox` 配合 `QSlider`（水平）实现。
  - `QSpinBox` 设置范围 100-10000，步长 100，单位后缀“ ms”。
  - `QSlider` 设置为 `Qt.Horizontal`，范围映射到相同数值区间。
  - 两者通过信号槽绑定：`spinBox` 值变化时更新 `slider` 位置，`slider` 滑动时更新 `spinBox` 值。
- **图片相似度阈值**：
  - 同样使用 `QSpinBox` + `QSlider`，范围 0-100，步长 1，单位“%”。
  - 下方放置说明文字 `QLabel`，内容固定。

#### 2.2.2 文本处理参数区域
- **文本去重阈值**：
  - 使用 `QSpinBox` + `QSlider`，范围 0-100，步长 1，单位“%”。
  - 下方放置说明文字。

#### 2.2.3 翻译结果显示设置区域
- **译文字体**：
  - 使用 `QFontComboBox`，该控件自动列出系统所有可用字体，并显示字体预览。
  - 也可使用普通 `QComboBox` 配合 `QFontDatabase` 手动填充字体名，但 `QFontComboBox` 更便捷。
- **字体大小**：
  - 使用 `QSpinBox` + `QSlider`，范围 8-72，步长 1，单位“pt”。
- **字体颜色**：
  - 使用 `QPushButton`，按钮上显示当前颜色块（如使用样式表设置背景色）和颜色文本（如“白色 #FFFFFF”）。
  - 点击按钮弹出 `QColorDialog`，用户选择颜色后，更新按钮显示并保存颜色值。
- **背景透明度**：
  - 使用 `QSpinBox` + `QSlider`，范围 0-100，步长 1，单位“%”。
  - 下方放置说明文字。

### 2.3 底部状态与按钮
- 状态标签：`QLabel`，初始文本“状态: 等待启动”，运行中显示“● 翻译运行中，参数已锁定”，暂停时显示“翻译已暂停”。
- “上一步”按钮：`QPushButton`，始终可点击（非运行态）或置灰（运行态）。
- “开始翻译” / “暂停翻译” 按钮：同一个按钮，根据状态切换文本。初始为“开始翻译”，运行态变为“暂停翻译”。

## 3. 数据绑定与配置管理

### 3.1 配置读取
- 界面初始化时，通过 `ConfigManager` 读取当前配置，填充所有控件。
- 若配置项不存在，使用默认值填充（需求中已定义默认值）。

### 3.2 控件与配置的对应关系
| 配置项 | 控件 | 数据类型 |
|--------|------|----------|
| `ocr.interval_ms` | 截图间隔 spinbox | int |
| `ocr.image_threshold` | 图片相似度阈值 spinbox | int |
| `ocr.text_threshold` | 文本去重阈值 spinbox | int |
| `display.font_family` | 字体下拉框 | str |
| `display.font_size` | 字体大小 spinbox | int |
| `display.font_color` | 颜色按钮 | str (如 "#RRGGBB") |
| `display.bg_opacity` | 透明度 spinbox | int |

### 3.3 实时预览（可选）
- 字体选择时，可通过信号 `currentFontChanged` 更新一个预览标签（如“示例文本”）的字体，但需求中仅提到可选，可不实现。

### 3.4 值验证与修正
- 所有 `QSpinBox` 设置 `setRange` 和 `setSingleStep` 自动保证范围。
- 若用户手动输入超出范围的值，spinbox 会自动修正为最近的有效值。
- 颜色选择对话框返回有效颜色，无需额外验证。

## 4. 状态管理

### 4.1 运行/暂停状态切换
- 定义枚举状态：`Idle`（空闲）、`Running`（运行中）、`Paused`（暂停）。
- 界面根据状态更新：
  - **空闲**：所有控件启用，底部按钮为“开始翻译”，“上一步”可点击。
  - **运行中**：所有控件禁用（`setEnabled(False)`），按钮为“暂停翻译”，“上一步”禁用。
  - **暂停**：所有控件恢复启用，按钮为“开始翻译”，“上一步”可点击。

### 4.2 状态标签文本更新
- 根据状态设置标签文本，并可能使用不同颜色（如运行中显示绿色圆点）。

### 4.3 状态保持与恢复
- 当用户点击“开始翻译”进入运行态后，若关闭程序再启动，应重新进入空闲态（因为后台线程未启动）。配置已保存，但状态不持久化。

## 5. 与后台线程的联动

### 5.1 启动翻译
- 点击“开始翻译”时（空闲态）：
  1. 收集当前所有控件的值，调用 `ConfigManager.save_config()` 持久化。
  2. 通过主窗口或全局控制器，启动截图线程和OCR/翻译线程（传入最新配置）。
  3. 创建并显示悬浮窗（若尚未创建）。
  4. 将界面状态切换为运行中。
  5. 记录日志。

### 5.2 暂停翻译
- 点击“暂停翻译”时（运行态）：
  1. 调用线程控制器的 `pause()` 方法，暂停截图循环和OCR/翻译任务处理。
  2. 界面状态切换为暂停（控件启用，按钮文本恢复“开始翻译”）。
  3. 状态标签更新为“翻译已暂停”。

### 5.3 恢复翻译
- 在暂停态点击“开始翻译”：
  1. 调用线程控制器的 `resume()` 方法。
  2. 界面切换回运行态。

### 5.4 停止翻译（退出程序）
- 关闭主窗口时，应调用线程控制器的 `stop()` 方法，等待线程结束，然后退出。

## 6. 与悬浮窗的联动

### 6.1 悬浮窗创建
- 悬浮窗应在第一次点击“开始翻译”时创建，并保持单例。
- 可由主窗口持有悬浮窗实例，或通过全局单例管理。

### 6.2 配置更新通知
- 当用户在运行参数界面修改显示配置并点击“开始翻译”（或“应用”）时，配置已保存。但悬浮窗需要实时反映新样式。
- 可在点击“开始翻译”前（空闲态）就应用新样式，或通过信号通知悬浮窗重新读取配置。
- 设计：在界面点击“开始翻译”时，除了保存配置，还可以发送自定义信号 `displayConfigChanged()`，悬浮窗连接该信号并在槽中更新样式。

### 6.3 暂停时隐藏/显示
- 暂停翻译时，悬浮窗可继续保持显示，显示最后一次翻译结果，或可选择清空。需求未明确，建议保持显示，以便用户参考。

## 7. 异常处理

### 7.1 启动失败
- 若因窗口句柄失效等原因导致后台线程无法启动，应弹出警告框，并停留在空闲态。
- 例如：截图线程初始化时发现窗口无效，可发送信号给主窗口，主窗口提示用户重新选择窗口。

### 7.2 配置保存失败
- 配置管理器写入文件失败时，应记录日志并提示用户（但通常不常见）。

## 8. 与其他模块的交互

| 模块 | 交互方式 |
|------|----------|
| 配置管理器 | 读取/保存配置 |
| 主窗口控制器 | 启动/暂停/停止后台线程 |
| 悬浮窗 | 通过信号通知配置变更 |
| 截图线程 | 通过线程控制器间接控制 |
| OCR/翻译线程 | 通过线程控制器间接控制 |

## 9. 实现注意事项

- 所有滑块与spinbox的联动应避免信号循环：可使用 `blockSignals(True)` 临时阻断，或使用 `valueChanged` 与 `sliderMoved` 区分。
- 字体下拉框若使用 `QFontComboBox`，其显示的是字体名称，可直接用于创建 `QFont`。
- 颜色按钮的样式更新：可重设按钮的样式表，如 `background-color: #RRGGBB;`，同时设置适当文本颜色保证对比度。
- 状态标签的圆点可使用HTML格式：`<font color='green'>●</font> 翻译运行中`。

## 10. 后续扩展预留

- 若需增加更多参数（如OCR语言切换、代理设置），可在对应分组框内添加新控件。
- 若需支持配置预设（保存多组配置），可增加“保存预设”和“加载预设”按钮，但当前需求未包含。
